## Colors
- Hexadecimal colors:
	- *f has the greatest intensity* and *0 has the lowest intensity*.
	- We type it as `#ffffff` 
- RGB colors:
	- `rgb(0,0,0)`
	- `rgba(0,0,0,1)`: last parameter is opacity, the colour is fully opaque. It ranges from 0 to 1.

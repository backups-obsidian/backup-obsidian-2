## CLI Tools
- `duf`: better `df`
	- `brew install duf`
- `fzf`: fuzzy finder
	- `brew install fzf`
	- Fuzzy finding in a path: `vim some_dir/**`
	- If you have the zsh integration with fzf then you can just press tab: `kill -15 <press_tab>`
- `ctop`: container `htop`
	- `brew install htop`
- `lazydocker`: docker cli tool
